# Themis Release 下载说明

## 快速开始

1. 下载与你系统匹配前缀的全部文件（如 windows-x86_64-*）到**同一文件夹**。
2. 打开 ***-README.md**，将 ***-env.example** 复制为 **.env** 并填入 Azure Speech（及可选 LLM）密钥。
3. 运行 *-themis-tray / *-themis-tray.exe；用 *-themis-cli doctor 确认配置。

> **未创建 .env 时，听写为 Mock 模式（假字幕），不能用于真实场景。**
